# 更新日志

## [v0.3.0] - 2026-05-15

### Changed

- 不再在插件侧构建索引，改为直接消费仓库提供的预构建 `elysian-realm-index.json`
- 索引架构调整为 “预构建默认索引 + 本地自定义 overlay” 分层模型：默认索引只读，本地关键词可写并在运行时合并
- 查询流程从旧平铺结构下按关键词全表扫描的 `O(n)`，调整为倒排索引查表定位候选（平均 `O(1)`）后再按 `last_updated` 选图，在资源规模增大时可显著减少查找开销并提升响应速度

## [v0.2.0] - 2026-04-30

### Added

- 新增管理员标识列表与非管理员可用指令列表，支持更细粒度的权限控制
- 新增 `/强制获取乐土攻略` 指令，可删除损坏或被误改的本地仓库后重新浅克隆远端攻略仓库
- 新增 Git 代理配置，支持预设代理地址和自定义代理地址
- 新增分组配置结构，将配置整理为仓库同步、自动回复、权限控制三类 object 配置

### Changed

- `/获取乐土攻略` 与 `/更新乐土攻略` 在执行长时间操作前会先返回进度提示，减少无响应观感
- 更新前会自动同步本地仓库 origin 远端地址到当前代理配置，便于切换不同代理后继续拉取
- README 与配置说明已同步更新，明确新增图片时当前仍需手动补充关键词

### Fixed

- 改进异常处理，避免单次命令执行异常导致插件整体不可用
- 改进群聊/私聊白名单与非管理员指令放行逻辑的兼容性
